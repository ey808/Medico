# Medico Chatbot

**Medico** is an AI-powered medical chatbot integrated into a **Django** application. Using a **Basic RAG** architecture, Medico provides reliable guidance on health topics, focusing on **breast cancer** and **influenza (grippe)**. It helps users understand symptoms, prevention measures, and general health advice.

---

## Features

- Answers health-related questions with accurate and context-aware responses.
- Focused on **breast cancer** and **influenza (grippe)** information.
- Uses **Basic RAG (Retrieval-Augmented Generation)** to fetch relevant medical knowledge.
- Built with **Django** for a secure and scalable web interface.
- Provides clear guidance for symptoms, prevention, and general health tips.

---

## Tech Stack

- **Backend:** Django  
- **AI Engine:** Basic RAG (Retrieval-Augmented Generation)  
- **Frontend:** HTML/CSS/JS
